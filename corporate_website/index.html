SENSMOVE
  • Le projet
  • L'équipe

SensMove est une semelle intelligente qui permet à l’utilisateur de visualiser la force exercée par ses appuis plantaires dans le temps. Grâce à une interface web dédiée, podologue, kinésithérapeutes ou ostéopathes peuvent suivre leurs patients dans leurs rééducations ou améliorer leurs examens posturaux.

La paire de semelle SensMove est accompagnée d’une application mobile iOS accessible sur le Apple Store grand public et d’une interface web disponible pour les professionnels de santé afin qu’ils puissent suivre les données de leurs patients.
